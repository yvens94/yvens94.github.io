Image classifier, handwriting digit recognition MNIST

Second project out of 5 to complete the Data science and Statistics program at the Massachusset Institute of Technology. This project consisted to compare many different approach to find which one perfom the best on our usecase. Similar to all the project in this program I had to build the algorithms from Scratch using numpy and scipy but in some specific case Scikit-Learn was allowed in some cases in PART 1: The different approaches started with regression, Using closed form and gradien descent to find the best parameters, softmax regression with augmented features, One vs Rest support vector machines, Multiclass SVM, Radial Basis Function SVM, polynomial Kernel, and cubic transformation in Part 2: We used Neural Network, convolutional neural network using pytorch
